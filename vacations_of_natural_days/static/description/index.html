Vacations Of Natural Days

This module was written to extend the functionality of hr_holidays to support vacations of natural days

Installation

To install this module, you need to:

  • hr_holidays

Configuration

To configure this module, you need to:

  • In Settings->Companies->Any Company
  • Check field, vacations_calendar_days
  • After, in Calendar Default (field displayed) add one attendance in Saturday and Sunday

Usage

To use this module, you need to:

  • If there is more than one company, position us where we are going to take a vacation.
  • New Leaves.
  • If the steps have been followed, they will count weekend days as working days and will adjust the calculation of vacation days to that amount.
